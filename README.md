# WEB IDE - AWEB IDE

> A Vue.js project

## 命令

- 安装
  ```sh
  npm i
  ```

- 开发前端页面

  ```sh
  npm run dev
  ```

- 打包前端页面

  ```sh
  npm run build
  ```
  
- 前端分析报告

  ```sh
  npm run build --report
  ```

- 前端单元测试
  
  ```sh
  npm run unit
  ```

- 前端集成测试

  ```sh
  npm run e2e
  ```
  
- 前端集成测试

  ```sh
  npm test
  ```
